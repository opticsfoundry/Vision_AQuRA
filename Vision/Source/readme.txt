Installing instructions for Vision

Install the files to harddisk C into the respective directories (or adapt drive letter in Vision.cfg).
Set screen mode to 1152x864 pixel, 32bit.
Start Vision.exe. 
Load eg. c:\Lithium\data\O1dec05\soliton.dat
Click on data point.

If files are missing install Borland C++ 5.02.
If necessary get TVicPort 3.1 demo from http://www.entechtaiwan.com/tools.htm.
